## By Kevin, Jake, and Karan

### Setup Instructions

1. cd server

2. npm i

3. download the .env file and put it in the server directory

3. cd ../client

4. npm i

5. npm run dev

6. open another terminal and cd into server folder

7. node server.js

8. go to http://localhost:5173/
